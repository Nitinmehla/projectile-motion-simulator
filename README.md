# Projectile Motion Simulator

This project simulates projectile motion using Python.

It calculates:

- Time of flight
- Maximum height
- Range of projectile

## Inputs

- Initial velocity (m/s)
- Angle of projection (degrees)

## Formula Used

Physics equations for projectile motion under gravity.
